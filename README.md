# HAND_GESTURE_RECOGNITION
This project implements a Hand Gesture Recognition System using deep learning techniques, developed and trained in Google Colab. The model is capable of predicting hand gestures from input images and verifying them against a trained dataset to provide accurate classification results.
 1. Open the file in google colab.
 2. Train the dataset given in google colab.
 3. Check the accuracy rate.
 4. Upload any hand gesture and check the prediction is correct or not.
